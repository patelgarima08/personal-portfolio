Project 1: Stone , paper and scissor game.


Description:I have built and stone paper scissor game using HTML,CSS and javascript in this game Stone crushes Scissors ,Scissors cuts Paper and Paper covers Stone.This game is played between two players. In this digital version, a player competes against the computer. Upon choosing one of the three options, the computer randomly selects its choice. The outcome is then displayed, announcing a win, loss, or draw based on the standard game rules. The game is fast-paced and easy to play, providing fun moments with a strategic twist!

Technologies Used: HTML, CSS and javascript..

Project 4: Stopwatch


Description : I have build a spotwatch using HTML , CSS and javascript.This watch shows different features.This stopwatch allows users to start, stop, and reset time with the click of a button. Key features include:

Start/Stop Functionality: Users can start and stop the stopwatch with a single click.
Reset Button: Resets the time back to zero for a fresh start.
Time Display: Shows hours, minutes, seconds, and milliseconds for precise tracking.
